# MERN Calendar App

A full-stack MERN (MongoDB, Express, React, Node.js) calendar application with authentication, CRUD operations, and email sending.

## Tech Stack

- **Frontend:** React + TypeScript + Vite
- **Backend:** Node.js + Express + TypeScript
- **Database:** MongoDB + Mongoose
- **Auth:** JWT + bcryptjs
- **Email:** Nodemailer + Mailtrap
- **Calendar:** react-big-calendar
